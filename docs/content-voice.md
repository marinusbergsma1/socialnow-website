# SocialNow Content Voice Guide

> **Gebruik:** Copywriters, AI-tools, marketingcampagnes, social media managers

---

## Kernprincipes

### 1. Data boven adjectieven
Zeg **nooit** "geweldige resultaten" — zeg altijd **"340% engagement boost"**.

| ❌ Slecht | ✅ Goed |
|-----------|---------|
| "Wij leveren hoge kwaliteit" | "98% van onze klanten verlengt hun contract" |
| "Snelle levering" | "Live binnen 3 weken" |
| "Professioneel team" | "50+ projecten, 4.9/5 rating" |
| "Wij zijn experts" | "React 19, TypeScript, Cloudflare Edge" |

### 2. Terminal-esthetiek in headings
Gebruik systeem-terminologie voor headings en labels. Dit versterkt de tech/AI positionering.

**Goedgekeurde STATUS-labels:**
- `SCANNING` — Data-analyse fase
- `ENCODING` — Branding/strategie fase
- `BUILDING` — Development fase
- `DEPLOY_OPS` — Lancering fase
- `LIVE` — Actief/lopend
- `VERIFIED` — Gecertificeerd resultaat
- `CERTIFIED` — Kwaliteitskeurmerk
- `SYNC` — Samenwerking/integratie
- `OPTIMAL` — Topprestatie

### 3. Taal: NL voor gesprek, EN voor techniek
- Nederlandse bodytekst voor alle klantcommunicatie
- Engelse labels voor technische termen (STATUS, system nodes, tech specs)
- Nooit gemengde zinnen: "Wij leveren top-tier results" ❌

### 4. Urgentie zonder wanhoop
- Schaarste moet **echt** zijn (deadline: "t/m eind maart 2026")
- Geen nep-urgentie ("Nog maar 3 plekken!" zonder bewijs)
- Altijd een specifieke datum of limiet noemen

## Headline Formules

### Hero Headlines
```
THE NEXT GENERATION OF [DIENST]
STOP MET GOKKEN OP [PIJNPUNT].
[GETAL]+ [RESULTAAT] IN [TIJDSBESTEK]
VAN [PROBLEEM] NAAR [OPLOSSING] IN [GETAL] STAPPEN
```

**Voorbeelden:**
- "THE NEXT GENERATION OF AUTOMATION"
- "STOP MET GOKKEN OP SUCCES."
- "2M+ BEREIK IN 6 MAANDEN"
- "VAN LEAD NAAR KLANT IN 3 STAPPEN"

### CTA Headlines
```
CLAIM JE [SPECIFIEK DING]
LANCEER JOUW [MERK/WEBSITE/SYSTEEM]
PLAN EEN [GRATIS/VRIJBLIJVENDE] [CALL/DEMO]
```

**Voorbeelden:**
- "CLAIM JE GRATIS CONTENT PAKKET"
- "LANCEER JOUW MERK"
- "PLAN EEN GRATIS KENNISMAKINGSCALL"

### Subheadlines
Bodytekst in h2/h3 positie — mag iets menselijker:
- "Geen bureau. Een systeem."
- "Wij vertalen rauwe ambitie naar visuele monumenten."
- "Onze strategie is gecodeerd voor totale marktdominantie."

## Social Media Copywriting

### Instagram Caption Structuur
```
[HOOK — max 1 regel, getal of vraag]

[BODY — 2-3 zinnen, data-gedreven]

[CTA — specifiek en direct]

[HASHTAGS — max 5, op nieuwe regel]
```

**Voorbeeld:**
```
340% meer engagement in 3 maanden.

Niet door meer te posten — maar door slimmer te posten.
AI-gedreven content die de algoritmes begrijpt en benut.

Wil je weten hoe? Link in bio.

#AIMarketing #SocialNow #ContentAutomation #DigitalMarketing #Amsterdam
```

### LinkedIn Post Structuur
```
[STELLING of FEIT — 1 zin, met getal]

[CONTEXT — 2-3 zinnen, wat het probleem is]

[OPLOSSING — 2-3 zinnen, hoe SocialNow het aanpakt]

[BEWIJS — 1-2 zinnen, client resultaat of data]

[VRAAG aan de lezer — voor engagement]

[CTA — subtiel]
```

### TikTok Script Structuur
```
0-3s: HOOK — Schreeuw of getal
3-10s: PROBLEEM — Herkenbaar scenario
10-25s: OPLOSSING — Snelle demonstratie
25-30s: CTA — Eén specifieke actie
```

## E-mail Marketing

### Onderwerpregel Formules
- `[GETAL] reden waarom [DOELGROEP] [PROBLEEM] heeft`
- `Gratis [WAARDE] voor [DOELGROEP] — t/m [DATUM]`
- `Hoe [KLANT] [RESULTAAT] behaalde in [TIJDSBESTEK]`

**Voorbeelden:**
- "3 redenen waarom jouw website leads misloopt"
- "Gratis website demo t.w.v. €10.000 — t/m 31 maart"
- "Hoe RAVEG 340% meer engagement behaalde in 2 maanden"

### E-mail Structuur
```
ONDERWERP: [Formule hierboven]
PREVIEW TEKST: [Tweede hook — aanvullend, niet herhalend]

Groet [NAAM],

[OPENING — 1 zin, direct to the point]
[PROBLEEM — 2 zinnen]
[OPLOSSING — 2-3 zinnen, SocialNow USP]
[BEWIJS — 1 klant case of statistiek]
[CTA BUTTON — groot, groen, 1 actie]

[URGENTIE REGEL — deadline of limiet]

Met groet,
[NAAM]
SocialNow | Amsterdam
```

## Verboden Woorden & Zinnen

### Absoluut verboden
- "Passie voor ons vak"
- "Wij denken mee"
- "Full-service"
- "Maatwerk oplossingen"
- "Uw succes is onze missie"
- "Wij staan voor u klaar"
- "Kwaliteit staat voorop"
- "Jarenlange ervaring"

### Gebruik spaarzaam (alleen met data)
- "De beste" → Moet bewezen worden: "De #1 qua engagement voor [niche]"
- "Professioneel" → Zeg liever: "ISO-gecertificeerd" of "4.9/5 rated"
- "Snel" → Zeg liever: "Live in 3 weken"
- "Goedkoop" → Nooit gebruiken, vervang door "hoge ROI"

## Tone Spectrum

| Context | Toon |
|---------|------|
| Hero headlines | Direct, powerful, UPPERCASE |
| Service beschrijvingen | Technisch, data-driven, beknopt |
| Client testimonials | Authentiek, resultaat-gefocust |
| FAQ | Helder, eerlijk, geen salespraat |
| E-mail marketing | Direct, persoonlijk, urgentie |
| Social media | Energetisch, visueel-ondersteunend |
| Offerte / voorstel | Professioneel, gestructureerd, cijfer-gedreven |

## AI Prompt Templates

### Voor Claude/ChatGPT — Social media post schrijven
```
Schrijf een Instagram caption voor SocialNow over [ONDERWERP].
Stijl: direct, data-gedreven, terminal-esthetiek.
Toon: geen bullshit, alleen harde feiten en resultaten.
Geen: "passie voor ons vak", "maatwerk", "wij denken mee".
Include: een statistiek of specifiek resultaat.
Max: 150 woorden + 5 hashtags.
Eindig met een directe CTA.
```

### Voor Claude/ChatGPT — E-mail schrijven
```
Schrijf een sales e-mail voor SocialNow voor [DOELGROEP].
Doel: [ACTIE — bv. kennismakingscall boeken].
USP: gratis [ACTIE] t.w.v. [WAARDE], t/m [DATUM].
Toon: direct, professioneel, geen salespraat.
Bewijs: gebruik het resultaat van [KLANT]: [RESULTAAT].
Max: 200 woorden.
```

### Voor Claude/ChatGPT — Advertentie copy schrijven
```
Schrijf 3 varianten ad copy voor SocialNow voor [PLATFORM].
Formaat: [FORMAAT — bv. single image, carousel, video script].
Doelgroep: [PERSONA — bv. "Scale-up founder, 28-40, Amsterdam"].
Pijnpunt: [PROBLEEM].
USP: [OPLOSSING].
CTA: [ACTIE].
Stijl: geen jargon, direct, resultaat-gericht.
```
